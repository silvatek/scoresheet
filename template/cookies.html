Overview

The scoresheet web application uses a minimal set of cookies to make the site functional. We only use "first-party" cookies, which cannot be used to track you across multiple sites.
More information about cookies in general, and the EU restrictions on their use, is available at https://gdpr.eu/cookies/

Cookie Details

_csrf
These cookies are used to prevent Cross-Site Request Forgery attacks, where another website you visit tries to submit data to this one without your knowledge. There may be multiple _csrf cookies.
gameHistory
The game history cookie is used to store references to the games and lists that you have accessed recently, allowing you to quickly return to those items.
scoresheetStyle
We may occasionally experiment with different visual styling options for the website, and the scoresheetStyle cookie is used to capture which style should be used.
